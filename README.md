# Sistema de Predicción de Lluvia
![Texto alternativo](https://github.com/AlvarezNelson5623/Proyecto-Inteligencia-Artificial/blob/main/Banner.png)

Este repositorio contiene el código y la documentación del proyecto de predicción de lluvia. El proyecto tiene como objetivo desarrollar diferentes modelos de clasificación para predecir la ocurrencia de lluvia, basado en variables como la temperatura, humedad, velocidad del viento, entre otros, con el fin de anticipar condiciones climáticas.
